# FastAPI Mini Project

## وصف المشروع
مشروع صغير باستخدام FastAPI + MySQL + bcrypt لبناء API لإدارة المستخدمين.

## التقنيات المستخدمة
- Python 3.10+
- FastAPI
- MySQL
- Pydantic
- bcrypt

## تثبيت المشروع
```bash
git clone <repo-link>
cd FastAPI-miniProject
python -m venv venv
# على لينكس / ماك
source venv/bin/activate
# على ويندوز
venv\Scripts\activate
pip install -r requirements.txt
